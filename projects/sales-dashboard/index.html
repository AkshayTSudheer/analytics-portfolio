Sales Performance Dashboard

Interactive Power BI dashboard built to analyze sales trends, regional performance, and revenue growth.

Problem Statement

Businesses lack visibility into sales performance across regions, products, and time periods. This dashboard helps stakeholders quickly identify revenue drivers, declining segments, and growth opportunities.

Dataset

  • Source: Sample Superstore Dataset
  • Fields: Orders, Sales, Profit, Region, Category
  • Size: ~10,000 records

Methodology

  • Data cleaning using Power Query
  • Created calculated measures using DAX
  • Designed KPI metrics (Revenue, Profit, Growth)
  • Built interactive filters and drilldowns

Key Insights

  • Top-performing region contributed 40% of total revenue
  • Technology category showed highest profit margin
  • Seasonal spikes observed in Q4 sales
  • Discounts negatively impacted profitability
